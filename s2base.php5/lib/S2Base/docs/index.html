S2Base.PHP5 プロジェクトへようこそ！

S2Base.PHP5プロジェクトでは、S2Container.PHP5 と S2Dao.PHP5 を用いた開発を支援するコマンドツールを作成します。

ダウンロード

    s2base.php5-1.0.0-rc3.zip
    s2container.php5-1.1.1 と s2dao-1.1.0 を含めています。
  • 2006-06-03
    s2base.php5-1.0.0-rc2.zip
    s2container.php5-1.1.0 と s2dao-1.1.0-rc3 を含めています。
  • 2006-05-08
    s2base.php5-1.0.0-rc1.zip
    s2container.php5-1.1.0 と s2dao-1.1.0-rc2 を含めています。

動作環境

PHP-5.1 以降で動作します。以下の環境で開発・動作確認しています。

  • WindowsXP : php-5.1.4-Win32
  • Debian GNU/Linux 3.1 : php-5.1.4
  • CentOS 4.3 : php-5.1.4
  • Mac OS X 10.4.7 : php-5.1.4
